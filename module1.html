Module 1: Introduction to Computers & Devices

A computer is a machine that can perform various tasks like processing data, running programs, and connecting to the internet.

Desktop: A desktop computer is a stationary unit designed for personal use.

Laptop: A laptop is a portable computer that combines all the components of a desktop in one compact form.
